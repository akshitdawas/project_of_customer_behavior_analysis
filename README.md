Data Analytics Project

Overview

This project demonstrates a complete Data Analytics workflow, starting from data collection and preprocessing to visualization and reporting. The objective was to extract meaningful insights from raw data using Python, SQL, and Power BI, and present findings through a professional report and presentation.

---

Project Workflow

1. Data Loading

- Imported dataset using Python.
- Verified data structure and data types.
- Performed initial data inspection.

2. Data Cleaning

- Handled missing values.
- Removed duplicate records.
- Corrected inconsistent data formats.
- Managed outliers where necessary.

3. Exploratory Data Analysis (EDA)

- Analyzed distributions and trends.
- Identified relationships between variables.
- Created visualizations for better understanding.
- Generated business-focused insights.

4. SQL Analysis

Executed analytical queries using:

- PostgreSQL
- SQL Server

Key analyses included:

- Aggregations
- Filtering and sorting
- Group By operations
- Joins
- Window Functions
- KPI calculations

5. Power BI Dashboard Development

Created an interactive dashboard containing:

- KPI Cards
- Trend Analysis Charts
- Category-wise Performance
- Filters and Slicers
- Interactive Visualizations

6. Reporting

Prepared a detailed project report including:

- Problem Statement
- Methodology
- Data Analysis
- Insights
- Recommendations

7. Presentation

Developed a professional presentation using Gamma AI to summarize:

- Project Objectives
- Key Findings
- Dashboard Overview
- Business Recommendations

How to Run

Python Analysis

1. Install required libraries:
   
   pip install pandas numpy matplotlib seaborn

2. Open and run the Python notebook/script.

SQL Analysis

1. Import the dataset into PostgreSQL, MySQL, or SQL Server.
2. Execute the SQL queries available in the project.

Power BI Dashboard

1. Open the ".pbix" file in Power BI Desktop.
2. Refresh the dataset connection.
3. Explore dashboard insights interactively.
